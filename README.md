# 🐱🐶 Cats vs Dogs Classification using SVM

## 📌 Overview
This project implements a **Support Vector Machine (SVM)** model to classify images of **cats and dogs** using the Kaggle dataset.

The model uses **image preprocessing + feature flattening** to convert images into numerical vectors before training.

---

## 🚀 Features
- Image preprocessing using PIL
- Grayscale conversion & resizing
- Feature vector creation
- SVM classification using scikit-learn
- Model saving & prediction support

---

## 📂 Dataset
Dataset used: Kaggle Dogs vs Cats  
Images are labeled as:
- `cat.0.jpg`, `cat.1.jpg`, ...
- `dog.0.jpg`, `dog.1.jpg`, ...

⚠️ Dataset is not included in this repository due to size.
